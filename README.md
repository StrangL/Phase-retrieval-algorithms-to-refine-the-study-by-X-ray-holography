# Phase-retrieval-algorithms-to-refine-the-study-by-X-ray-holography

Курсовая работа на тему: "Алгоритмы восстановления фазы для уточнения исследования методом рентгеновской голографии".

Реализован алгоритм восстановления для одномерного случая.

Показано, что в зависимости от разного количества известной фазы, скорость сходимости алгоритма будет отличаться.
